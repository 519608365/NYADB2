# NYADB2
NYADB2是一个Go编写的数据库.

它主要实现了:
+ 数据库的可靠性
+ 事务处理
+ MVCC
+ 并发索引

我创建NYADB2的目的是为了学习, NYADB2的模型简单, 实现简洁, 整个代码+注释大概约7000行.

毫无疑问, NYADB2也还有许多并不完善的地方, 我将会持续的完善她, 以后将会有NYADB3, 4, 5, ...
